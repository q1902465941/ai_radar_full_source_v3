# 数据说明

本目录记录策略研究使用的数据口径。不要在这里保存 API key、secret、账户截图或任何敏感信息。

## 允许使用的数据

- Binance 主网公开行情。
- 雷达扫描快照。
- 纸面交易记录。
- 已关闭持仓记录。
- 回放样本。
- 人工整理的研究样本。

## 数据必须标注

每份数据集至少说明：

- 来源。
- 时间范围。
- 字段含义。
- 是否为真实成交、纸面成交或回放样本。
- 是否包含手续费。
- 是否包含滑点。
- 是否包含失败样本。

## 样本分层

研究时必须区分：

- 样本内：用于提出和调试策略。
- 样本外：用于验证策略是否还能工作。
- 纸面样本：用于验证执行闭环。
- 影子实盘样本：只观察真实盘口，不真实下单。

不要把回放样本、纸面样本和真实成交样本混在一起统计胜率。
